# Генератор сообщений для клиентов (Message Generator)

## Описание

Функция для быстрого создания персонализированных сообщений для отправки потенциальным клиентам.

**Цель:** Упростить партнёрам отправку сообщений с предложением юридических услуг, предоставляя готовый текст с высокой конверсией.

---

## Файлы

- `bot/keyboards/keyboards.py` — добавлена кнопка "💌 Отправьте это и получите ₽₽₽"
- `bot/handlers/how_to_earn.py` — обработчик кнопки и генерация сообщения
- `bot/states/states.py` — новое состояние FSM `waiting_for_recipient_name`

---

## Как это работает

```
Пользователь нажимает "📖 Инструкция, как заработать"
                    ↓
Нажимает "💌 Отправьте это и получите ₽₽₽"
                    ↓
Бот просит ввести имя получателя и окончания
                    ↓
Пользователь вводит: "Александр" или "Александра"
                    ↓
Бот генерирует персонализированное сообщение
                    ↓
Пользователь копирует и отправляет клиенту
```

## Технические детали

### Состояние FSM

```python
class RevenueStates(StatesGroup):
    waiting_for_recipient_name = State()  # Состояние для имени получателя
```

### Обработчики

#### 1. `send_message_handler` (callback_query)

**Callback:** `earn_send_message`

**Описание:** Запрашивает имя получателя

**Код:**
```python
@router.callback_query(F.data == "earn_send_message")
async def send_message_handler(callback_query: CallbackQuery, state: FSMContext) -> None:
    text = (
        "Мы выверили сообщение, которое дает максимальную конверсию! 💰\n\n"
        "Введите <b>Имя получателя</b>, и если вы девушка — измените окончания, пример: «искал» на «искала»:\n\n"
        "<i>Пример: Александр / Александра</i>"
    )
    await callback_query.message.answer(text, parse_mode="HTML")
    await state.set_state(RevenueStates.waiting_for_recipient_name)
    await callback_query.answer()
```

---

#### 2. `process_recipient_name` (message)

**Состояние:** `RevenueStates.waiting_for_recipient_name`

**Описание:** Генерирует и отправляет сообщение

**Код:**
```python
@router.message(RevenueStates.waiting_for_recipient_name)
async def process_recipient_name(message: Message, state: FSMContext) -> None:
    recipient_name = message.text.strip()
    await state.update_data(recipient_name=recipient_name)
    
    generated_message = MESSAGE_TEMPLATE.format(name=recipient_name)
    
    # Отправляем для просмотра
    await message.answer(
        f"<b>Ваше сообщение для отправки:</b>\n\n{generated_message}",
        parse_mode="HTML"
    )
    
    # Отправляем в блоке кода для копирования
    await message.answer(
        f"<code>{generated_message}</code>",
        parse_mode="HTML"
    )
    
    await message.answer(
        "📋 Нажмите на сообщение выше, чтобы скопировать его, и отправьте вашему клиенту!\n\n"
        "Удачи в продажах! 💰",
        reply_markup=get_main_menu_keyboard()
    )
    
    await state.clear()
```

## Особенности

- ✅ **Простота использования** — всего 2 шага до готового сообщения
- ✅ **Персонализация** — имя получателя вставляется автоматически
- ✅ **Удобное копирование** — сообщение в блоке кода копируется одним нажатием
- ✅ **Гендерная адаптация** — пользователь сам выбирает окончания при вводе
- ✅ **Высокая конверсия** — текст выверен и оптимизирован
- ✅ **Автоматическая очистка состояния** — после генерации состояние сбрасывается
