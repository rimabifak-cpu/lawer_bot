# Инструкции по запуску админ панели и телеграм бота

## Быстрый запуск

Для быстрого запуска админ панели и телеграм бота выполните следующие шаги:

1. Запустите скрипт `start.bat` (Windows):
   ```
   start.bat
   ```

2. Скрипт запустит:
   - Админ панель на порту 8001
   - Телеграм бот в отдельном окне

## Проверка работы

- Админ панель доступна по адресу: http://127.0.0.1:8001
- Бот обрабатывает входящие сообщения и отправляет уведомления о новых пользователях

## Ручной запуск

Если скрипт не работает, запустите каждый компонент отдельно:

### 1. Запустите админ панель

Откройте командную строку и выполните:
```bash
cd C:/Users/HONOR/Documents/law_bot
python admin_panel/app.py
```

### 2. Запустите телеграм бот

Откройте новую командную строку и выполните:
```bash
cd C:/Users/HONOR/Documents/law_bot
python run_bot.py
```

## Проверка отправки сообщений

Для проверки работы отправки сообщений используйте curl:
```bash
curl -X POST "http://127.0.0.1:8001/api/messages/direct" -H "Content-Type: application/json" -d "{\"telegram_id\": 5093303797, \"content\": \"Тестовое сообщение\"}"
```

## Настройка уведомлений о новых пользователях

Для того чтобы уведомления о новых пользователях были отправлены в чат админов, нужно:

1. Добавить бота в чат "ЮР - партнёры"
2. Получить правильный ID чата, используя @userinfobot или команду /getid
3. Обновить значение ADMIN_CHAT_ID в файле .env
