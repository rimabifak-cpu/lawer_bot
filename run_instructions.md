# Инструкции по запуску бота с новой функциональностью

## Шаг 1: Настройка переменных окружения

1. Откройте файл `.env` в текстовом редакторе
2. Найдите строку `ADMIN_CHAT_ID=-1001234567890`
3. Замените значение на ID чата "ЮР - партнёры" (см. инструкции в `get_chat_id_instruction.md`)
4. Сохраните изменения

## Шаг 2: Запуск бота

Если бот уже запущен, остановите его (Ctrl+C), затем запустите снова:

```bash
cd law_bot
python run_bot.py
```

## Шаг 3: Проверка функциональности

1. В Telegram откройте чат с ботом @legaldecision_bot
2. Перейдите в раздел "Партнёрский профиль"
3. Выберите "Заполнить/обновить мои данные"
4. Введите информацию о профиле
5. После сохранения профиля проверьте, что уведомление пришло в чат "ЮР - партнёры"

## Возможные проблемы

- Если уведомления не приходят в чат "ЮР - партнёры", проверьте:
  - Правильность ID чата в файле `.env`
  - Наличие бота в этом чате
  - Права бота на отправку сообщений в чате
