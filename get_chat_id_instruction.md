# Получение ID чата "ЮР - партнёры"

Чтобы бот @legaldecision_bot мог отправлять карточки профиля в чат "ЮР - партнёры", вам нужно получить ID этого чата и обновить файл `.env`.

## Способ 1: Использование бота @userinfobot

1. Добавьте бота [@userinfobot](https://t.me/userinfobot) в чат "ЮР - партнёры"
2. Перешлите любое сообщение из этого чата боту @userinfobot
3. Бот пришлет информацию, в которой будет ID чата (начинается с -100)

## Способ 2: Использование бота @getmyid_bot

1. Откройте диалог с [@getmyid_bot](https://t.me/getmyid_bot)
2. Добавьте бота в чат "ЮР - партнёры"
3. Бот пришлет вам ID чата

## Способ 3: Использование Telegram Web/Telegram Desktop

1. Откройте чат "ЮР - партнёры"
2. Нажмите на название чата, чтобы открыть информацию
3. В адресной строке браузера (если используете Telegram Web) будет URL с ID чата

## После получения ID чата

1. Откройте файл `.env` в корневой директории проекта
2. Найдите строку `ADMIN_CHAT_ID=`
3. Замените значение на полученный ID чата (например: `ADMIN_CHAT_ID=-1001234567890`)
4. Сохраните файл

После этих действий бот будет отправлять карточки профиля в чат "ЮР - партнёры" при заполнении пользователем информации о профиле.
