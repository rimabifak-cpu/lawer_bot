# Отправка карточек профиля в чат "ЮР - партнёры"

## Описание изменений

Была реализована функциональность, позволяющая боту @legaldecision_bot отправлять карточки профиля в чат "ЮР - партнёры" при заполнении пользователем информации о профиле.

## Что было изменено

1. В файле `bot/handlers/profile.py`:
   - Добавлен импорт настроек из `config.settings`
   - Обновлена логика отправки уведомлений о новых/обновлённых партнёрских профилях
   - Теперь уведомления отправляются в чат, ID которого указан в переменной `ADMIN_CHAT_ID`

2. В файле `.env`:
   - Обновлена документация для переменной `ADMIN_CHAT_ID`
   - Добавлены инструкции по получению ID чата "ЮР - партнёры"

## Настройка

Для корректной работы функциональности необходимо:

1. Получить ID чата "ЮР - партнёры" (см. инструкции в `get_chat_id_instruction.md`)
2. Обновить значение переменной `ADMIN_CHAT_ID` в файле `.env`
3. Перезапустить бота

## Как получить ID чата

См. подробные инструкции в файле `get_chat_id_instruction.md` или запустите скрипт `get_chat_id_helper.py`.
